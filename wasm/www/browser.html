How it works

Your browser sends keypresses. The chain does everything else.

Execution

Deterministic runtime

Every game tick is a state transition executed identically by every validator. Integer-only fixed-point math — no floating point, no divergence. The same inputs always produce the same game state, block after block.

Infrastructure

Validators render your frames

Player inputs are submitted as extrinsics. The game state is ticked on-chain every block. When your browser needs a frame, a validator raycasts the scene, composites sprites, and streams back raw pixels. Your device just paints them to screen.

Substrate

Custom pallet architecture

The DOOM engine is a Substrate pallet — a runtime module that sits alongside staking, governance, and token pallets. Each player gets an independent on-chain game state. Up to 32 concurrent players per chain.
